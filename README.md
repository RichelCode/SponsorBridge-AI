# SponsorBridge AI

## Overview
SponsorBridge AI is a voice-enabled RAG chatbot designed to help international students explore companies in the United States that have historical sponsorship activity.

The system collects public data, processes it, and allows users to ask questions through text or voice.

## Goals
- Help international students understand companies better
- Provide insights from public sponsorship-related data
- Build a fully functional AI system using real-world tools

## Features (Planned)
- Web scraping and data collection
- Data cleaning and storage
- Retrieval-Augmented Generation (RAG)
- Speech-to-text interaction
- API backend
- User interface
